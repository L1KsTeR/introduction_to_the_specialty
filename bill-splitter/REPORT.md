## 1. 📋 Описание системы

**bill-splitter** — минимальное веб-приложение для расчёта разделения счёта в ресторане с учётом чаевых.

## Проблема
При посещении ресторана компанией часто возникает необходимость быстро посчитать, сколько должен заплатить каждый человек с учётом чаевых. Калькуляторы на телефонах требуют ручного ввода формулы, что неудобно.

## Решение
Веб-приложение с простым интерфейсом:
- Ввод суммы счёта
- Ввод процента чаевых
- Ввод количества человек
- Мгновенный расчёт с детализацией

## Целевая аудитория
Люди, которые ходят в рестораны компаниями и хотят быстро разделить счёт.

### Компоненты системы

| Компонент | Файл | Ответственность |
|-----------|------|-----------------|
| **Routes** | `app.py` | HTTP-эндпоинты, обработка запросов |
| **Validators** | `validators.py` | Проверка входных данных |
| **Logic** | `logic.py` | Чистая бизнес-логика расчёта |
| **Frontend** | `templates/`, `static/` | Пользовательский интерфейс |

### Поток данных

1. Пользователь вводит данные в форму (HTML)
2. JavaScript отправляет POST-запрос на `/api/calculate`
3. Flask принимает JSON в `app.py`
4. `validators.py` проверяет данные (положительные числа, обязательные поля)
5. `logic.py` считает результат
6. JSON-ответ возвращается на клиент
7. JavaScript отображает результат

## 3. ️ Стек технологий

### Backend
- **Python 3.13** — язык программирования
- **Flask 3.1.3** — микро-фреймворк для веб-приложений
  - *Почему Flask:* лёгкий, простой для учебных проектов, хорошо подходит для API

### Frontend
- **HTML5** — разметка
- **CSS3** — стилизация
- **Vanilla JavaScript** — клиентская логика (fetch API)
  - *Почему без фреймворков:* минимальный проект, не требует React/Vue

### Инфраструктура
- **Docker** — контейнеризация
- **GitHub Actions** — CI/CD
- **pytest** — тестирование
- **Ruff** — линтинг
- **Black** — форматирование
- **Bandit** — статический анализ безопасности
- **Trivy** — сканирование зависимостей

## 4. 🚀 Инструкции по запуску

### Вариант 1: Локальный запуск

```bash

git clone https://github.com/L1KsTeR/introduction_to_the_specialty.git
cd introduction_to_the_specialty/bill-splitter/backend

# 2. Создай виртуальное окружение
