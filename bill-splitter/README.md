# 💰 Калькулятор разделения счёта + чаевые

Минимальное веб-приложение (Frontend + Backend), демонстрирующее структуру, валидацию и обработку ошибок.

## 🛠 Стек
- **Backend:** Python 3, Flask
- **Frontend:** HTML5, CSS3, Vanilla JS (`fetch` API)
- **Архитектура:** MVC-подобное разделение (маршруты → валидация → бизнес-логика → представление)
