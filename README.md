# QuickNotes — Минималистичное веб-приложение для быстрых заметок

## Идея проекта
Создать легковесное веб-приложение для мгновенного создания и хранения личных заметок. Главная цель — убрать лишние шаги: пользователь открывает страницу и сразу начинает печатать. Заметки сохраняются автоматически, без регистрации. Проект решает проблему перегруженности аналогов избыточным функционалом и подходит для студентов, фрилансеров и всех, кому нужно быстро фиксировать мысли, ссылки или задачи.

## Основной функционал (MVP)
- Создание, редактирование и удаление заметок
- Автосохранение в `localStorage` при каждом изменении
- Мгновенный поиск по заголовку и тексту
- Добавление цветных тегов (Учёба, Работа, Личное, Срочное)
- Адаптивный интерфейс (mobile-first)

## Архитектура
Клиентское SPA-приложение без серверной части на старте:
- **Данные**: хранятся в `localStorage` в формате JSON. Структура заметки: `{ id, title, content, tags, createdAt, updatedAt }`
- **Состояние**: управляется через простой Pub/Sub паттерн. При изменении данных все подписанные компоненты перерисовываются
- **UI**: разделён на три независимые зоны — левая панель (список + поиск), центральный редактор, верхняя панель (теги + настройки)
